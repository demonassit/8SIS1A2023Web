Todos los formularios que se manejan dentro de la web utilizan metodos de envio para cada uno de los datos, que nos sirven para las solicitudes de información (request) los mas comunes son:
GET: Este metodo se encarga de enviar la informacion por la url o la salida estandar del navegador, esto propicia que se pueda visualizar toda la información del formulario en la url
POST: este metodo se encarga de enviar la informacion por la url; pero de forma oculta, lo que ofrece cierto nivel de seguridad, este metodo se usa comunmente para insertar un nuevo registro dentro de los sistemas de información
